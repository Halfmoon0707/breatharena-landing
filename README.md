# BreathArena Landing Page

**India's First Premium Sleep Tape**

## 🚀 Coming Soon Page

Landing page for BreathArena.com - launching India's first premium mouth tape brand for better sleep.

## 📦 Tech Stack

- HTML/CSS/JavaScript (simple & fast)
- Email capture → Google Sheets integration
- Deployed on Vercel
- Custom domain: BreathArena.com

## 🏗️ Project Structure

```
breatharena-landing/
├── index.html          # Main landing page
├── styles.css          # Styling
├── script.js           # Email capture logic
├── assets/             # Images, logos
└── README.md
```

## 🎯 Features

- ✅ Coming Soon hero section
- ✅ Email waitlist capture
- ✅ Social media links
- ✅ Mobile responsive
- ✅ Auto-deploy via Vercel

## 📝 Setup

1. Clone the repo
2. Connect to Vercel
3. Add custom domain (BreathArena.com)
4. Configure email capture (Google Sheets API)
